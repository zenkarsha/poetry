# 仲夏夜

我試著舔嗜<br/>
她的身體不自覺的扭動、抽動<br/>
發出有些高頻的呻吟<br/>
一陣一陣<br/>
慢慢的變得溼潤<br/>
右手輕輕搓揉、撫摸著她<br/>
那因抽動挺起的乳房<br/>
身體的反應更大了<br/>
呻吟中帶著持續的嬌喘<br/>
<br/>
裡頭溢出今夏的夜露<br/>
身體的漫舞漫出一整灘的滋潤<br/>
舌在齒間打轉<br/>
我喜歡她整齊潔白的牙<br/>
和此時濃密香甜的吻<br/>
正試著壓抑那無法克制的合聲<br/>
<br/>
那是一整個夏夜的激情<br/>
我愛上她乳房上露出的汗珠<br/>
和那闔起眼時<br/>
性感的笑容
